# ConsultSavvy — AI-Powered Management Consulting Framework

> MCA Final Year Project | AI-Integrated Web Application

## Overview

ConsultSavvy is a full-stack AI-powered management consulting portal 
that enhances the efficiency and effectiveness of organizational 
consulting processes. Built as an MCA final year project, it 
demonstrates practical integration of large language models into 
real-world business decision-making workflows.

The system combines live AI intelligence with human consultant 
oversight — ensuring every recommendation is reviewed, approved, 
and signed off by a qualified professional before reaching the client.

## Objectives

- Develop an AI-powered system that enhances the efficiency and 
  effectiveness of management consulting processes
- Leverage AI technologies to provide data-driven insights and 
  recommendations for improving organizational performance 
  and decision-making
- Explore the potential of AI in automating repetitive tasks and 
  augmenting human consultants' capabilities

## Core Modules

**Executive Dashboard**
Real-time KPI monitoring, pipeline visualization, AI-generated 
insight cards, and performance analytics across business units

**AI Strategy Advisor**
Natural language consulting interface powered by Claude AI — 
context-aware responses, conversation memory, consultant notes, 
and session summarization

**Strategic Analysis Suite**
AI-generated SWOT analysis, Porter's Five Forces scoring, and 
KPI benchmarking with radar chart visualization

**AI Report Builder**
Six report templates with dynamic AI-generated content, structured 
JSON parsing, consultant sign-off workflow, and print-ready output

**Task Automation Engine**
Task queue management, AI vs human assignment tracking, automation 
health monitoring, and weekly activity dashboard

## Human-in-the-Loop Features

- Decision log with consultant approve and reject controls
- Client sentiment tracker assessed by the consultant
- Consultant field notes private workspace
- Mandatory consultant sign-off on every generated report
- Client profile card persisted across the full session
- Session timer tracking active consulting time

## Tech Stack

- Frontend: HTML5, CSS3, Vanilla JavaScript
- AI Integration: Anthropic Claude API (claude-sonnet-4-20250514)
- Data Visualization: Chart.js 4.4.1
- Typography: Google Fonts — DM Sans, DM Serif Display
- Architecture: Single Page Application with dynamic page routing
- Deployment: GitHub Pages

## Key Design Decisions

The platform is deliberately designed as a human-AI collaboration 
tool rather than a fully automated system. AI handles data 
processing, pattern recognition, and report drafting — while human 
consultants retain full control over strategic decisions, client 
communication, and final deliverables.
